# Crime

A web-based crime data exploration tool.  
[Visit the site](https://atlanta-crime-nextjs.vercel.app/)!

## Data

- [Crimes](https://www.atlantapd.org/i-want-to/crime-data-downloads)
- [Atlanta neighborhood lists](https://www.atlantaga.gov/government/departments/city-planning/office-of-zoning-development/neighborhood-planning-unit-npu/npu-by-neighborhood)
- [Atlanta neighborhood polygons](https://gis.atlantaga.gov/dpcd/rest/services/OpenDataService/FeatureServer/3)
- [Offense listings](https://www.atlantapd.org/home/showpublisheddocument/2881/637062879602730000)
